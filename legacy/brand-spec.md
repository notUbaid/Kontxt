## Logo

"Kon" in deep burgundy (#8B004A), "txt" in dark navy (#13265C). Minimalist, modern, no icon.

## Font Stacks

- Display: `-apple-system, BlinkMacSystemFont, 'SF Pro Display', system-ui, sans-serif`
- Body: `-apple-system, BlinkMacSystemFont, 'SF Pro Text', system-ui, sans-serif`
- Mono: `'JetBrains Mono', 'IBM Plex Mono', ui-monospace, Menlo, monospace`

## Posture

- Developer-tool UI / documentation browser
- Warm neutral background (Murrey), deep burgundy primary accent, dark navy secondary
- Clean editorial feel with utility density
- Glassmorphism for sticky nav (backdrop-blur)
- Hairline borders, minimal shadows
- Primary accent (burgundy) for CTAs, active nav, sign-in button
- Secondary accent (dark navy) for mode switcher, secondary UI signals, phase labels
